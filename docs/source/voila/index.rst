
Voila usage 
===============================

We use Voila to render Jupyter notebooks into web applications.  Voila is
developed under the Jupyter team. By using Voila, we can deploy our apps at
Binder and Dokku (Materials Cloud). This is the tutorial for the usage of Voila.
